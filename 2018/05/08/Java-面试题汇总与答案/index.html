基本概念

  • 操作系统中heap和stack的区别

stack的空间由操作系统自动分配和释放，heap的空间由程序员手动的分配和释放，通常使用new关键字分配，Java中，若只是声明一个对象，则在栈中分配内存地址空间，new之后才在堆中分配地址空间
